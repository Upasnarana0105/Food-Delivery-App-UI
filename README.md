# Food-Delivery-App-UI
A modern food delivery app UI design prototype created in Figma showcasing user-friendly navigation, product listing, and order flow.
live Prototype: [View on Figma] (https://www.figma.com/proto/nYmO3h355FYtwhkpl7arcL/Untitled?node-id=0-1&t=vYlPcQGKqW38ZKC0-1)

---

Features
1- Interactive prototype with real-time flow
2- Clean and minimal user interface 
3- Easy navigation from home to checkout 
4- Food listing and order screens with visual hierarchy

---

Tool used
Figma- UI/UX design
Prototyping & User Flow Design
Typography & Color Theory

---

About
Designed to demostrate user-centered design principles for a food delivery application.
Created as part of a UI/UX learning project to explore layout balance and usability.
